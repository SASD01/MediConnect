# MediConnect — Patient Portal (UTB)
MediConnect is a patient health portal that lets users find available doctors, request medical appointments, and view their latest clinical record in a secure and simple experience.

## Key features

Appointment booking: search availability, schedule, cancel, and (optionally) reschedule visits
Doctors directory: filter by specialty, location, modality, and date
Latest clinical record: read the most recent visit summary (and attachments if enabled)
Secure access: OTP (One-Time Password, one-time code) login and protected actions
Audit trail: logging of sensitive actions (view record, create/cancel appointments)

## Tech stack
Backend: FastAPI (Python)
Mobile: Flutter
Web: Astro
Database & Auth: Supabase (PostgreSQL + Auth)
